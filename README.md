# Wheel Tracker

A full-stack application for tracking wheel strategy trades, built with React (frontend) and FastAPI (backend).

## Deployment

### Development Mode (with hot-reload)

**Windows (PowerShell):**
```powershell
.\deploy.ps1 dev
```

**Linux/Mac/Git Bash:**
```bash
./deploy.sh dev
```

### Production Mode

**Windows (PowerShell):**
```powershell
.\deploy.ps1 prod
# or simply
.\deploy.ps1
```

**Linux/Mac/Git Bash:**
```bash
./deploy.sh prod
# or simply
./deploy.sh
```

## Access

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs

For more details, see [DEPLOYMENT.md](./DEPLOYMENT.md).
